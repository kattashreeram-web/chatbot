# Mini Todo Chatbot

## Requirements
Python 3.8+

## Setup

1. Create virtual environment (optional but recommended):
   python -m venv venv
   source venv/bin/activate  (Mac/Linux)
   venv\Scripts\activate     (Windows)

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   python app.py

4. Open browser:
   http://127.0.0.1:5000
